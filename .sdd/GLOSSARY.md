# Glossary

## SDD

Spec-Driven Development. A development approach where the specification is the source of truth for implementation.

## Spec

A structured description of a feature, architecture change, integration, bugfix, or research item.

## Task

A small implementation unit derived from a spec and plan.

## ADR

Architecture Decision Record. A document explaining an important architectural decision.

## Spec Tree

A hierarchical view of all specifications in the repository.

## Project Memory

Markdown-based repository context used by humans and AI agents.
