# SDD Workspace

This folder is the repository-native project memory for SpecFlow for VSCode.

Markdown files are the source of truth. JSON files under `.sdd/STATE` are rebuildable indexes only.

## Main folders

- `SPECS/` — specifications and implementation plans.
- `DECISIONS/` — architecture decision records.
- `TEMPLATES/` — reusable SDD templates.
- `AGENTS/` — instructions for LLM agents.
- `STATE/` — generated indexes and cache.
