# Design

## Main UI Areas

1. Activity Bar item: SpecFlow.
2. Spec Tree View.
3. Spec Dashboard WebView.
4. Task Board WebView.
5. Spec Creation Wizard (native VSCode flow).
6. Product Map WebView.

## Spec Creation Wizard

The wizard should collect:

- title;
- type;
- priority;
- owner;
- problem;
- goal;
- acceptance criteria;
- whether to create a starter task.

## Product Map

The Product Map should show:

- feature cards;
- status;
- priority;
- task progress;
- quick actions to open dashboard/task board/source files.

## Localization

WebViews should support English and Russian labels based on VSCode UI language.
