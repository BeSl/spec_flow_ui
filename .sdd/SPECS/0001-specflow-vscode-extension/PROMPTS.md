# Prompts

## Implementation Prompt Template

Use `.sdd/TEMPLATES/prompt.implementation.template.md`.

## Review Prompt Template

Ask the model to compare implementation against:

- `SPEC.md`
- `PLAN.md`
- `TASKS.md`
- `ARCHITECTURE.md`
- `REVIEW.md`

## Test Prompt Template

Ask the model to create tests only for reviewed changes and acceptance criteria.
